# Docker Learning Guide

A beginner-friendly, in-depth guide to learn Docker from scratch.

## Guide Structure

| # | File | What You'll Learn |
|---|------|-------------------|
| 1 | [01-what-is-docker.md](./01-what-is-docker.md) | What Docker is, why it exists, and the problem it solves |
| 2 | [02-core-concepts.md](./02-core-concepts.md) | Images, Containers, Dockerfile, Volumes, Networks — with real-life analogies |
| 3 | [03-installation.md](./03-installation.md) | How to install Docker on your machine |
| 4 | [04-essential-commands.md](./04-essential-commands.md) | Every important Docker command you'll use daily |
| 5 | [05-dockerfile-guide.md](./05-dockerfile-guide.md) | Writing Dockerfiles — instructions, best practices, examples |
| 6 | [06-docker-compose.md](./06-docker-compose.md) | Multi-container apps with docker-compose.yml |
| 7 | [07-volumes-and-networks.md](./07-volumes-and-networks.md) | Data persistence and container communication |
| 8 | [08-real-world-tips.md](./08-real-world-tips.md) | Tips, debugging, cleanup, and production practices |

## How to Use This Guide

- Read files **in order** (01 → 08)
- Each file builds on the previous one
- Code snippets are included — try them out as you go
- After completing this guide, you'll be ready for the **mini project** (dockerizing a full-stack app with MySQL + Express + React)

## Prerequisites

- Basic understanding of terminal/command line
- Know what a server and database are (at a high level)
- That's it! No prior Docker knowledge needed.
